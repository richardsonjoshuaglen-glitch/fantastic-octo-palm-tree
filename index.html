Rigging CG Helper v11

This is the v11 build placeholder. In the actual implementation, JavaScript computes pulley drops and hook position automatically from the fixed 20' chokers and loop lengths.

Features in v11:

  • 4-point mode with Pulley + Inverted Baskets (auto pulley Z, auto hook XYZ)
  • 20' basket loops enforced with wrap/bend allowances
  • 20' top chokers enforced (hook solved as highest sphere-sphere intersection)
  • Output includes tape-ready offsets and forces
  • Plan and Elevation drawings with true contact points
